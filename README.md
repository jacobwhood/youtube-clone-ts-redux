# youtube-clone-ts-redux
App to Search and watch YouTube video using React, Redux, TypeScript

helpful article found here: https://hackernoon.com/react-with-typescript-and-webpack-654f93f34db6

## Getting Started
```
npm i
````

development:
```
npm start-dev
```

production:
```
npm start
```
